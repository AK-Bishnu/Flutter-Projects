# Flutter Projects Portfolio

A collection of Flutter applications showcasing my journey from beginner to advanced development.

## Projects (from the beginning of my journey to current):

A classic Snake game implemented in Flutter.

### 2. Tetris Game
A simple Tetris game built with Flutter.

### 3. Learn C Programming
A C Programming learning project with multiple stages, levels, and animations to help beginners understand programming concepts.

### 4. Quick Enhancer
A Flutter application that quickly enhances images by adjusting image pixel values.

### 5. Sound Bound
A Flutter-based network audio streaming application that allows devices to connect over a local network and transmit microphone audio between host and clients in real time, with synchronized DJ lighting effects using device flashes.
