# Quick Enhancer 📸

A Flutter application that quickly enhances images by adjusting image pixel values. This app provides both automatic and manual enhancement options to improve your photos with professional-quality results.

## Features 🚀
- **Image Analysis**: Analyze your images to identify areas needing improvement in brightness, contrast, saturation, highlights, and shadows.
- **Auto Enhance**: One-click automatic enhancement that optimally adjusts all image parameters.
- **Manual Edit**: Fine-tune your images with individual sliders for brightness, contrast, saturation, highlights, and shadows.
- **Before/After Comparison**: Toggle between original and enhanced versions to see the improvements.
- **Real-time Preview**: See changes instantly as you adjust parameters.
